# ディレクトリブラウザ 設計

---

## パンくずリスト（Breadcrumb）

ディレクトリを深く潜った際に現在地を示すパンくずをナビゲーションバー下部またはコンテンツ上部に表示する。

### 表示イメージ

```
サンプルNAS  >  Videos  >  映画  >  2024
```

### 仕様

- 各セグメントはタップ可能で、タップするとその階層まで一気に戻る
- 表示しきれない場合は左側を省略して末尾（現在地）を常に表示する
  - 例: `...  >  映画  >  2024`
- 先頭セグメントはNASの表示名（例: `サンプルNAS`）を表示し、タップするとトップフォルダに戻る
- トップフォルダより上の階層へは遡れない（startPathがルートでない場合もNASルートへは戻れない）
- 横スクロール可能にして全階層を辿れるようにする

### 配置

- ナビゲーションバーの下、ファイル一覧の上に固定表示
- 縦持ち・横持ち共通

### 実装方針

- 現在のパスを `[String]` の配列で管理（`FileBrowserViewModel` が保持）
- パンくずの各セグメントは配列のインデックスに対応
- タップ時はそのインデックスまで配列を切り詰め、該当ディレクトリを再読み込み
- `ScrollView(.horizontal)` + `HStack` で実装し、自動で末尾にスクロール

## ディレクトリブラウザ全体レイアウト

```
┌─────────────────────────────────────┐
│  ← サンプルNAS          並び替え  検索  │  ← ナビゲーションバー
├─────────────────────────────────────┤
│  サンプルNAS > Videos > 映画 > 2024     │  ← パンくずリスト
├─────────────────────────────────────┤
│  📁 アクション      📁 SF           │
│  🎬 サンプル動画A       🎬 サンプル動画B    │  ← サムネイルグリッド
│  🎬 タイトルC       🎬 タイトルD    │
│           ...                       │
└─────────────────────────────────────┘
```

## ナビゲーションバー

| 要素 | 位置 | 説明 |
|---|---|---|
| 戻るボタン（←） | 左端 | 1つ上の階層へ。ルートではホーム画面へ |
| タイトル | 中央 | 現在のフォルダ名 |
| 並び替えボタン | 右から2番目 | 並び替え基準を選ぶメニュー（名前 / 追加日時 / 更新日時 / サイズ） |
| 検索ボタン | 右端 | 現在のディレクトリ内をファイル名で絞り込み |
