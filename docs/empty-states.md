# 空状態のUI設計

各画面で「コンテンツがない」状態のUI。
アイコン + 説明文 + アクションボタン の構成で統一する。

---

## 各画面の空状態

### ホーム画面（接続先なし）

```
        🗄
   接続先がありません
ネットワークサービスを追加して
  動画・画像を閲覧しましょう

  [+ 接続先を追加]
```

- アイコン: SF Symbols `network`
- ボタン: NAS追加シートを開く

---

### フォルダが空

```
        📁
   このフォルダは空です
```

- アイコン: SF Symbols `folder`
- ボタンなし（操作できることがないため）

---

### メディアリスト（登録なし）

```
       🔖
   リストがありません
動画・画像を選択して
  リストに追加できます
```

- アイコン: SF Symbols `list.bullet`
- ボタンなし（ブラウザから操作するため）

---

### 視聴履歴一覧（履歴なし）

```
        🕐
   視聴履歴がありません
動画や画像を開くと
  ここに履歴が表示されます
```

- アイコン: SF Symbols `clock`
- ボタンなし

---

### 検索結果が0件

```
       🔍
「○○」に一致するファイルは
   見つかりませんでした
```

- アイコン: SF Symbols `magnifyingglass`
- ボタンなし
- 「○○」は検索キーワードを動的に表示

---

## 共通デザイン仕様

- アイコンサイズ: 60pt
- アイコンカラー: グレー（`secondary` カラー）
- テキスト: 中央揃え、`secondary` カラー
- アクションボタン: あるものはプライマリボタンスタイル
- 縦方向: 画面中央（`Spacer` で上下均等）
