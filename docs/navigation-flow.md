# 画面遷移図

---

## 画面一覧

| 画面名 | 概要 |
|---|---|
| ホーム画面 | ソース一覧・リスト・履歴のルート画面 |
| 接続追加画面 | サービス種別選択・接続情報の入力・接続テスト（シート） |
| ディレクトリブラウザ | フォルダ・動画・画像ファイルの一覧（再帰遷移） |
| 動画プレイヤー | 動画再生・シーク・リスト追加 |
| 画像ビューアー | 画像の全画面表示・スワイプで前後切り替え |
| メディアリスト | 登録済みリストとリスト内ファイルの一覧 |
| 視聴履歴一覧 | 最近閲覧した動画・画像の一覧 |

---

## 全体遷移図

```
┌─ アプリ起動
│
└─ ホーム画面（タブバーのルート）
    │
    ├─ [タブ: ブラウザ]
    │    │
    │    ├─ [+ボタン] ──────────────→ NAS追加画面（Sheet）
    │    │                                └─ [保存/キャンセル] → ホームへ戻る
    │    │
    │    ├─ [NAS選択] ──────────────→ ディレクトリブラウザ（Push）
    │    │                                │
    │    │                                ├─ [フォルダ選択] → ディレクトリブラウザ（Push、再帰）
    │    │                                │
    │    │                                ├─ [動画選択] ──→ 動画プレイヤー（fullScreenCover）
    │    │                                │                    └─ [×閉じる] → ブラウザへ戻る
    │    │                                │
    │    │                                └─ [画像選択] ──→ 画像ビューアー（fullScreenCover）
    │    │                                                     ├─ [スワイプ左右] → 同フォルダ内の前後画像
    │    │                                                     └─ [×閉じる] → ブラウザへ戻る
    │    │
    │    └─ [ローカル選択] ──────────→ ディレクトリブラウザ（Push）
    │                                    └─ （NASと同様）
    │
    ├─ [タブ: リスト]
    │    │
    │    └─ メディアリスト
    │          └─ [リスト内ファイル選択] → 動画プレイヤー（fullScreenCover）
    │                                      └─ [×閉じる] → メディアリストへ戻る
    │
    └─ [タブ: 履歴]
         │
         └─ 視聴履歴一覧
               └─ [履歴選択] ────→ 動画プレイヤー（fullScreenCover、最終視聴位置から再生）
                                      └─ [×閉じる] → 視聴履歴一覧へ戻る
```

## 縦持ち / 横持ちによる遷移の違い

### 縦持ち（メイン）

`NavigationStack` によるスタック遷移。タブバーをボトムに配置。

```
ホーム画面
  └─ Push → ディレクトリブラウザ
               └─ Push → ディレクトリブラウザ（深掘り）
                            └─ fullScreenCover → 動画プレイヤー
```

### 横持ち

`NavigationSplitView` による2カラム表示。

```
左カラム（サイドバー）         右カラム（メイン）
  ホーム画面                →    ディレクトリブラウザ
  [NAS/ローカル選択で更新]        └─ fullScreenCover → 動画プレイヤー
```

## 遷移ごとの仕様

### ディレクトリブラウザへの遷移

- NASへの接続が未確立の場合、遷移前に接続処理を実行
- 接続中はスピナーを表示
- 接続失敗時はアラートを表示してホームへ留まる

### 動画プレイヤーへの遷移（fullScreenCover）

- 遷移元の情報（sourceID・filePath・開始位置）をViewModelに渡す
- 開始位置の優先順位: 視聴履歴の最終位置 > 先頭（0秒）
- プレイヤーを閉じると自動的に視聴履歴を更新する

### NAS追加画面（Sheet）

- 接続テスト成功後でないと保存できない
- キャンセルで変更を破棄してホームに戻る
